## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
This app is a sign up page simulation. Shows developers how to display a warning message to the user. We used three TextFields in the application. We get the username, password, and password confirmation with these TextField structures. Then there is a FilledButton for the user to register. This button does some checking. The first is that the username is not empty. The second is that the password is blank and the last is that the password and authentication match. If no errors are found in these checks, it warns the user that he is registered. If it finds an error, it displays these errors as a warning message to the user. We created the alert with UIAlertController. Then we added a cancel button with UIAlertAction. We integrated the button into the alert with the addAction method. Finally, we showed the warning screen to the user with self.present.
